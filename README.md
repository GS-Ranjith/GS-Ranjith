## 📌 Featured Projects

### 1. 🧠 DocuSense RAG — Intelligent Document Q&A
&gt; Production-grade Retrieval-Augmented Generation system for enterprise document analysis.

- **What it does:** Answers questions based on your private documents using vector search + LLM
- **Tech:** Python, LangChain, ChromaDB, Google Gemini 2.5 Flash, Gemini Embedding 001
- **Impact:** 100% grounded answers, zero hallucinations, full source attribution
- 🔗 [View Repository](https://github.com/GS-Ranjith/DocuSense-RAG)

### 2. 🤖 IntentFlow AI — Conditional LLM Router
&gt; Smart intent classification system that routes queries to specialized AI chains.

- **What it does:** Detects intent (Technical/General/Greeting) and routes to appropriate response pipeline
- **Tech:** Python, LangChain, Gemini 2.5 Flash, RunnableLambda
- **Impact:** Modular AI architecture with context-aware responses
- 🔗 [View Repository](https://github.com/GS-Ranjith/Conditional-LLM-Assistant)

### 3. 🖼️ VisionAI — Deep Learning Image Classifier
&gt; Real-time image classification web app powered by MobileNetV2.

- **What it does:** Classifies uploaded images into 1000 categories with confidence scores
- **Tech:** Python, Streamlit, TensorFlow/Keras, OpenCV, MobileNetV2
- **Impact:** Instant visual recognition without custom training
- 🔗 [View Repository](https://github.com/GS-Ranjith/SmartImage-AI-Deep-Learning-Image-Classifier)
